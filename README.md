# Example Node.js Express using ES2015

## Getting Started

First install the npm dependencies

`$ npm install`

## Start Project

This project use [nodemon](https://nodemon.io/) with the [babel-node](https://babeljs.io/docs/en/next/babel-node.html)

`$ npm start`

## Build the project

`$ npm run build`

## Test

`npm test`
